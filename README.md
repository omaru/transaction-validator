# Transaction Validator
Gets a list of transactions and validates whether  transaction numbers are duplicated and balances are ok per record.
## Installation
### Compile
``` mvn clean package```
### Run
``` java -jar target/transaction-validator-0.0.1-SNAPSHOT.jar```
